Semantic Markup

  • it is markup that related to the meaning of content
  • semantic elements are those that clearly describe their meaning to both the browser and the developer.

  • Semantic markup is the use of HTML tags to convey the meaning of the content.
  • It helps search engines and assistive technologies understand the content of a webpage.
  • It also helps developers to write more meaningful and maintainable code.
  • Screen readers use semantic markup to provide a better experience for users with disabilities.
  • improves user experience by providing a clear structure and hierarchy to the content.
  1. Semantic elements include: header, nav, main, article, section, aside, footer, etc.
  2. Non-semantic elements include: div, span, etc.
